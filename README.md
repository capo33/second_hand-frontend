## Usage

### Env Variables

Create `.env` file and add the following

```
PORT = 5000
```

### Install Dependencies (frontend & backend)

```
cd backend
yarn
cd frontend
yarn
```

### Run

```

# Run frontend (:3000)
cd frontend
yarn start

 # Run backend (:5000)
yarn run dev
```
